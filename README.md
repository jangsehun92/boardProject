# boardProject
Spring + maven + oracle 기반 CRUD 게시판

## Front-end
* Bootstrop3
* jQuery3

## Back-end
* Language : JAVA 1.8
* WEB Server : Apache
* WAS Server : Tomcat 8
* Framework : Spring 5.1.3
* build tool : Maven 3.5.1
* ORM : mybatis 3.4.6
* DB : Oracle 11g


## 구현 기능
* 게시글 CRUD
* 페이징
* 검색 기능
* 로그인 기능
* 이메일 인증 기능



-추후 SpringBoot를 이용하여 리펙토링
